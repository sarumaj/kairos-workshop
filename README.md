## Overview

This repository demonstrates how to build and release custom Kairos OCI (Open Container Initiative) images using CI/CD pipelines. It focuses on automating the creation of immutable OS images tailored for Kubernetes deployments.

### What This Repository Covers

- Building custom Kairos OCI images
- Integrating Kairos image builds into CI/CD pipelines
- Publishing and releasing custom immutable OS images

### What is Kairos?

[Kairos](https://kairos.io) is an open-source Linux toolkit designed to make immutability practical and flexible for Kubernetes deployments. It provides a cloud-native, distro-agnostic, immutable Linux meta-distribution with features like A/B system partitions, persistent data separation, and cloud-config-driven provisioning.
